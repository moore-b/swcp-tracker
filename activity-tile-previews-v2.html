Color Assignment Strategy: Varied Green/Teal Gradients

How It Works

Base Green (#56801E → #436229)
Base Teal (#4d8596 → #3d6b7a)
Varied Green (#5a8a5e → #4a7a4e)

Each activity gets a slightly different gradient based on its ID

Benefits

Visual variety without chaos
Stays within your theme
Consistent color family

Each tile looks unique but cohesive

Implementation Details

Color Range:

  • • Green: #56801E to #5a8a5e
  • • Teal: #4d8596 to #3d6b7a
  • • Mixed: #5a8a5e to #4a7a4e

Variation Method:

  • • Based on activity ID
  • • Slight hue/saturation shifts
  • • Maintains readability
